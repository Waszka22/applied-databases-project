Project Innovation Features
---

1. Conference Dashboard

An additional dashboard feature was added to the application.

This dashboard displays:
- total number of attendees
- most popular conference session
- company with the highest number of attendees

This feature uses SQL aggregate queries and improves the usability of the system by providing quick conference statistics and insights.

---

2. Delete Attendee

An additional attendee deletion feature was implemented.

This functionality:
- removes an attendee from the MySQL database
- removes attendee connections from the Neo4j database
- asks the user for confirmation before deletion

This innovation demonstrates CRUD operations across both relational and graph databases.
